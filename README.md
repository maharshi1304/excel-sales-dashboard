# 📊 Excel Sales Dashboard

## 📌 Project Overview
This project is a simple sales dashboard built using Microsoft Excel.

## 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## 📊 Features
- Sales analysis by Product
- Sales analysis by Region
- Interactive filtering using slicers
- Visual representation using charts
